# create-productflow-kit

Generate ProductFlow Kit starter applications.

```bash
npm create productflow-kit@latest my-app -- --template ai-saas --data jpa
```

## Templates

- `saas-admin`
- `ai-saas`
- `landing-page`
- `crm-admin`
- `content-platform`
- `knowledge-base`
- `workflow-approval`
- `spring-api`
- `vue-admin`

## Commands

```bash
productflow list
productflow preview ai-saas
productflow create my-app --template ai-saas --data jpa
productflow add module ai
```

## Data Layers

- `jpa` / `jpa-flyway`
- `mybatis` / `mybatis-plus`
